              el: '#app-5',
              data: {
                message: 'Hasta la vista!'

Extra Goodies

Shorthands

v-bind Shorthand


            <a v-bind:href="url"> ... </a>
          

            <a :href="url"> ... </a>

v-on Shorthand


            <a v-on:click="doSomething"> ... </a>
          

            <a @click="doSomething"> ... </a>

Computed Properties


            <div id="computed">
              <p>Message: {{ message }}</p>
              <p>Reversed message: {{ reversedMessage }}</p>
            </div>
          

            new Vue({
              el: '#computed',
              data: {
                message: 'Hello'
              },
              computed: {
                reversedMessage: function() {
                  return this.message.split('').reverse().join('');
                }
              }
            });

Message: {{ message }}

Reversed message: {{ reversedMessage }}

Class Binding


            <div :class="{ active: isActive }"></div>
          

            data: {
              isActive: true

Event Modifiers


            event.preventDefault();
          

            event.stopPropagation();
  • .stop
  • .prevent
  • .capture
  • .self

Stop the event propagation


              <a @click.stop="doSomething"></a>

Prevent default action


              <form @submit.prevent="doSomething"></form>

Modifiers are chainable


              <a @click.stop.prevent="doSomething"></a>

Attach to capture phase


              <div @click.capture="doSomething">...</div>

Only handle if event is on self


              <div @click.self="doSomething">...</div>

Key Modifiers

Only handle if the key pressed is enter


              <input @keyup.enter="doSomething">

Can also use keyCode


              <input @keyup.13="doSomething">
  • .enter
  • .tab
  • .delete
  • .esc
  • .space
  • .up
  • .down
  • .left
  • .right

Define custom key modifiers


              Vue.config.keyCodes.f1 = 112;
